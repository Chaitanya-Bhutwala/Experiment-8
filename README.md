Experiment 8: Numerical Differentiation
Overview:

This experiment focuses on estimating the derivative of a function using numerical methods. The objective is to understand how discrete data points can be used to approximate derivatives and analyze the accuracy of such approximations.

Methodology:
Definition of equally spaced x values with fixed step size
Use of given function values f(x)
Implementation of forward difference method:
Calculation of derivative using difference between consecutive values
MATLAB loop used for iterative computation
Results displayed in tabular format (x, f(x), derivative)

Results:
Successfully computed approximate derivatives using discrete data
Observed approximation error, especially at initial points
Accuracy improved for subsequent values
Identified that smaller step size increases accuracy
Understood limitations of forward difference method and need for higher-order methods
